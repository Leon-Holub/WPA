Hlavní atributy

  • required - povinné
  • disabled - zakáže
  • readonly - pouze pro čtení
  • placeholder - text, který se zobrazí, když je input prázdný
  • value - výchozí hodnota
  • name - jméno inputu
  • type - typ inputu
  • for - odkaz na příslušný input

Validace v HTML

  • required - povinné
  • minlength - minimální délka
  • maxlength - maximální délka
  • pattern - regulární výraz
  • min - minimální hodnota
  • max - maximální hodnota
  • step - krok
