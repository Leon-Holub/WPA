Atributy vázané k tagu

Tyto atributy jsou specifické pro určité HTML tagy:

  • <a href="https://example.com">Odkaz</a> - href určuje URL, na kterou odkaz směřuje.
  • <img src="obrazek.jpg" alt="Popis"> - src určuje cestu k obrázku, alt poskytuje alternativní text pro obrázek.
  • <input type="text" value="Ukázka"> - type určuje typ vstupního pole, value nastavuje výchozí hodnotu.

Globální atributy

Tyto atributy mohou být použity na jakémkoli HTML elementu:

  • class - Určuje jednu nebo více tříd pro element, které mohou být použity pro CSS styly.
  • id - Určuje unikátní id pro element, které může být použito pro CSS styly nebo JavaScript.
  • style - Určuje inline CSS styl pro element.
  • title - Určuje dodatečné informace o elementu, které se zobrazí jako tooltip při najetí myší.
  • data-* - Používá se k ukládání vlastních dat specifických pro stránku nebo aplikaci.
  • lang - Definuje jazyk obsahu elementu (např. lang="cs" pro češtinu).
  • hidden - Skryje element, aniž by ho odstranil z DOM.
  • draggable - Určuje, zda lze prvek přetahovat (draggable="true").
  • onclick - Definuje JavaScriptovou akci při kliknutí na jakýkoli prvek.

Stylové vlastnosti pomocí atributu style

  • Změna barvy textu: style="color: blue;"
  • Změna pozadí: style="background-color: yellow;"
  • Změna zarovnání - align: style="text-align: center;"
  • Změna zarovnání - float: style="float: right;"
  • Změna zarovnání - display: style="display: inline-block;"
  • Změna zarovnání - justify-content: style="display: flex; justify-content: space-between;"
  • Změna mezery mezi prvky - margin: style="margin: 20px;"
  • Změna vycpávky - padding: style="padding: 15px;"
  • Změna velikosti textu: style="font-size: 24px;"
  • Změna fontu - font family: style="font-family: Arial, sans-serif;"
  • Změna tloušťky fontu - weight: style="font-weight: bold;"
