Block a inline

Elementy v HTML můžeme rozdělit do dvou základních kategorií: block a inline.

Block

Block elementy zabírají celou šířku stránky a vždy začínají na novém řádku. Příklady block elementů:

  • <div>
  • <p>
  • <h1> až <h6>
  • <table>

Inline

Inline elementy zabírají jen tolik místa, kolik potřebují a nezačínají na novém řádku. Příklady inline elementů:

  • <span>
  • <a>
  • <img>
  • <strong> a <em>
